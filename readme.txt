# Application Android

Ce dossier contient le code source de l'application Android développée sous Android Studio.

## Description

L'application Android permet de :

- Visualiser les données détectées par le téléphone (UUID, Major, Minor remplis par l'utilisateur, et la pseudo adresse MAC).
- Contrôler les capteurs ou actuateurs (envoi des données vers l'ESP32 via Bluetooth Low Energy - BLE).

## Pré-requis

- Android Studio installé.
- Un émulateur Android ou un appareil physique pour les tests.
- Autorisation d'accès à Internet.
- Téléphone compatible avec **Bluetooth Low Energy (BLE)** pour interagir avec l'ESP32.

## Installation

1. Clonez ce dépôt ou téléchargez le projet dans Android Studio.
